ArcGIS Maps: Applying the Bookmark widget to a map layer

 

This article will guide you through several steps for applying the Bookmark widget to an ArcGIS Maps map layer.

 

1.  In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.

 

 

2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).

 

 

3. In the Map Layers list, search by name for the map layer you desire to apply the Bookmark widget.

 

 

4. Click the desired map layer to open the map layer's configuration form.

 

5. Navigate the map layer's form to the Widget Information tab.

 

6. In the Configuration text box, view the JSON used to construct the map layer.

 

JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.

 

7. Navigate the Configuration text box to where you would like to implement the widget. For this example, the code will be placed below the layer's visible parameter.

 

The bookmark widget allows users to navigate to configured extents. 

 

8. Navigate the Configuration text box to view the JSON widgets object. To configure the Bookmark widget, you need to specify the type of widget to create, and the bookmark parameter values that are required for the map to navigate to the bookmark's extent.

 

 The widget type is Bookmarks. To properly config the widget, you need to specify the individual bookmarks and their associated extent parameter values.

 

 

9. When the widget parameters are configured in the Configuration text box, click Update to apply the changes to the map layer.

 

You are now able to apply a Bookmark widget to an ArcGIS Maps map layer.
