ArcGIS Maps: Applying multi-variate symbology to a map layer


This article will guide you through several steps for applying the multi-variate symbology to an ArcGIS Maps map layer.


  1. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.



  2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).


  3. In the Map Layers list, search by name for the map layer you desire to apply multi- variate symbology.



  4. Click the desired map layer to open the map layer's configuration form.

  5. Navigate the map layer's form to the Layer configuration tab.

  6. In the Configuration text box, view the JSON used to construct the map layer.

    JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.

  7. Navigate the Configuration text box to where you would like to implement the widget. For this example, the code will be placed below the layer's opacity parameter.

  8. Multi-variate symbology uses visual variables to set stops for the multi-variate symbology.


    Visual variables allow symbols to change dynamically based on values contained in a numeric attribute field. There are four types of visual variable renderers that can be used to render features. These include, color, opacity, rotation, and size. The example below uses the size type to dynamically change the symbol size based on the feature's numeric attribute value in the count field.



    Visual variables are grouped into ranges of data values called stops. Between two and six stops can be applied in ascending order to the stops array. Each stop defines the mapping of values returned from the field to symbol sizes. Once the visual variables are defined, the size slider widget can be applied in the map layer's JSON configuration.


  9. Navigate the Configuration text box to view the JSON widgets object. To configure the multi-variate symbology, you need to apply multiple types of visual variables and their associated stops. For example, a multi-variate symbol could be configured to have size, color, and opacity visual variable types.


  10. When the multi-variate JSON parameters are configured in the Configuration text box, click Update to apply the changes to the map layer.


You are now able to apply multi-variate symbology to an ArcGIS Maps map layer. Download the multi-variate symbol JSON file.
