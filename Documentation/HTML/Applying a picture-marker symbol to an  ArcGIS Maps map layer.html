ArcGIS Maps: Applying a picture-marker symbol to an ArcGIS Maps map layer

 

A picture-marker symbol symbolizes points as graphics in an ArcGIS Maps map layer. Parameters exist for configuring the width, height, offset and angle of the graphic. For example, a picture marker symbol may be configured to display features using a graphic with a specific width and height values. Picture marker symbols may be applied to feature layers referenced in an ArcGIS Maps map layer.

 

This article will guide you through several steps for applying a picture-marker symbol to an ArcGIS Maps map layer.

 

1.  In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.

 

 

2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).

 

 

3. In the Map Layers list, search by name for the map layer you desire to apply a picture-marker symbol for displaying the layer's features.

 

 

4. Click the desired map layer to open the map layer's configuration form.

5. Navigate the map layer's form to the Widget Information tab.

6. In the Configuration text box, view the JSON used to construct the map layer.

JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.

7. Navigate the Configuration text box to view the title, dataType, and table parameters.

The title parameter represents the name of the layer that will appear in the ArcGIS Maps Layer List. The dataType parameter represents the ArcGIS Maps data type that will be used to represent the layer. The table parameter references the name of the table in the CMDB.

 

8. Navigate the Configuration text box to view the JSON symbol object. To configure the symbol, you need to specify the type of symbol to create, and the values that are required for configuring the symbol.

9. For example, if the symbol type is configured as picture-marker, you need to specify the URL location of the picture and the pixel width and height dimensions to display the picture as a marker symbol on the map.

 

The symbol's URL parameter references the image uploaded and stored in ServiceNow. The name of the file represents the image's sys_id value. To view the uploaded images, navigate to System UI > Images using the filter navigator. To obtain an images sys_id value, locate the image in the Images list, right-click the image preview, and choose Copy sys_id to copy the value to the clipboard.

 

10. When the symbol parameters are configured in the Configuration text box, click Update to apply symbology changes to the map layer.

 

You are now able to navigate to a map layer's configuration parameters and apply a picture-marker symbol to an ArcGIS Maps map layer.
