ArcGIS Maps: Display a non-parent map layer in a web browser


There are multiple reasons for displaying map layers without making them a parent or visible in the ArcGIS Map Directory. One reason might be you are configuring the map layer and don't want it cluttering up your map directory. Another reason might be that the map layer will only be a child layer referenced by a parent map layer. Whatever the reason, map layers should be viewed during configuration to ensure the layer is display data accurately.

This article will guide you through several steps for display a non-parent map layer in a web browser.

  1. In the Service Now Filter Navigator enter ArcGIS Map to view the ArcGIS Maps components.



  2. Click ArcGIS Map Directory to view the contents of the map directory.



  3. Click any of the map layers to view the map layer in the ArcGIS Map application.


  4. Open another tab in the web browser.


  5. Copy the URL from the displayed map layer, and paste it in the new browser tab, press Enter.


  6. The map layer displays in the new browser tab. Leave the browser tab open and navigate to the browser tab where the map layer was initially displayed.


  7. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.


  8. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).



  9. In the Map Layers list, search by name for the map layer you desire to view in a web browser.


  10. In the Map Layers list, click the map layer to open the map layer's configuration form.



    The ArcGIS Map map layer's Map Parent field is set to false.


    To view the map layer, you will need to obtain the map layer's sys_id value. The sys_id value tells the ArcGIS Maps application which layer to open in a web browser.


  11. In the Map Layer form's tool bar, right-click and choose Copy sys_id from the context menu.


  12. Navigate to the new browser tab displaying the map layer.


  13. In the URL paste the sys_id directly after the mapId%3D parameter. Press Enter.



The non-parent map is displayed in the web browser.


You are now able to display a non-parent map layer in a web browser.
