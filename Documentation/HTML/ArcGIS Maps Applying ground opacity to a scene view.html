ArcGIS Maps: Applying ground opacity to a scene view


The ArcGIS Maps application supports applying a ground opacity on layers in a scene view. Ground opacity enables features below ground to be visible in the scene view. For example, scene views containing underground utility features with ground opacity enabled would allow users to view the utility features through the ground layer and determine how they relate to features above ground.


This article will guide you through several steps for applying ground opacity to an ArcGIS Maps scene layer.


  1. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.



  2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).



  3. In the Map Layers list, search by name for the map layer you desire to apply a picture-marker symbol for displaying the layer's features.



  4. Click the desired map layer to open the map layer's configuration form.

  5. In the Configuration text box, view the JSON used to construct the map layer.

    JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.

  6. Navigate the Configuration text box to view the title, dataType, and table parameters.



    The viewType represents the type of view to be displayed in the ArcGIS Maps application. The title parameter represents the name of the layer that will appear in the ArcGIS Maps Layer List.


  7. Navigate the Configuration text box to the location where you would like to configure the ground opacity. To configure the ground opacity, you need to specify

    the navigationConstraint, and the opacity that are required for configuring ground opacity.

    The navigationConstraint property specifies if the scene view allows below ground navigation. By setting the type to none specifies there are no navigational constraints on the scene view. The opacity property specifies the level of opacity for the ground layer. This property can be any number between 0 and 1.

  8. When the ground JSON parameters are configured in the Configuration text box, click Update to apply the ground opacity to the map layer.
