Applying constraints to an ArcGIS Maps 2D map layer


This article will guide you through several steps for applying constraints to an ArcGIS Maps map layer.


  1. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.



  2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).



  3. In the Map Layers list, search by name for the map layer you desire to apply a constraint for displaying the map layer's features.


  4. Click the desired map layer to open the map layer's configuration form.

  5. Navigate the map layer's form to the Widget Information tab.

  6. In the Configuration text box, view the JSON used to construct the map layer.

    JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.

  7. Navigate the Configuration text box to view the title, datatype, and table parameters.



    The dataType parameter represents the ArcGIS Maps data type that will be used to represent the layer. The table parameter references the name of the table in the CMDB.

  8. For example, if the geometry type is configured as extent, you need to specify the min and max coordinate values for the constraint.


The constraints geometry specifies the type of geometry and the coordinates used to define the geometry. The scales represent the constraints min and max values.


10. When the constraints parameters are configured in the Configuration text box, click Update to apply the changes to the map layer.


You are now able to navigate to a map layer's configuration parameters and apply a constraint to an ArcGIS Maps 2D map layer.
