ArcGIS Maps: Creating a map layer

ArcGIS Map Layers are tables or collections of location-based data that are used to configure an ArcGIS Map. Data in a map layer is represented as points, lines, or polygons. Map Layers can be created from several different data source types. For example, any spreadsheet or table with records containing x and y field values can be used to create a map layer. When creating a map layer, the layer's symbology is configured in the configuration's JSON using one of the many renderers available and symbols.

This article will guide you through several steps for creating an ArcGIS Maps map layer.


  1. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.



  2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).



  3. In the Map Layers list, click New to open the form for creating a new map layer.



  4. Enter a unique name for the map layer.


  5. Click Active to make the map layer table available in the tableâ€™s directory and the Access Control Lists.


  6. For the Category field, enter ArcGIS Map.


  7. For the Subcategory field, enter Core Technology Services.



  8. Navigate the map layer's form to the Layer Configuration tab.

  9. In the Configuration text box, you will enter the JSON used to construct the map layer.

    JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.


  10. Begin constructing the map layer by entering the map layer's source JSON parameters.



The map layer's title is "Water". The title parameter value will appear as the layer's name in the ArcGIS Map's layer list widget. For this example, the dataType parameter is "gliderecord". A glide record represents a table containing fields and field values. The glide record table's x and y fields are used to display the geographic features on the ArcGIS Map. For this example, the glide record represents a table named u_utility_locations. The u_utility_locations table contains many different utility types. To only include a record of a specific type, an additional parameter can be included in the json that represents the name of the field containing the types and the desired type value. For example, "util_u_facility_type" is the name of the field containing the types and "Hydrant" is the desired type.

8. To configure the symbol, specify the type of symbol to create, and the values that are required for configuring the symbol.

For example, if the symbol type is configured as picture-marker, you need to specify the URL location of the picture and the pixel width and height dimensions to display the picture as a marker symbol on the map.


The symbol's URL parameter references the image uploaded and stored in ServiceNow. The name of the file represents the image's sys_id value. To view the uploaded images, navigate to System UI > Images using the filter navigator. To obtain an images sys_id value, locate the image in the Images list, right-click the image preview, and choose Copy sys_id to copy the value to the clipboard.


10. When the map layer's parameters are configured in the Configuration text box, click Update to apply the configuration JSON to the map layer.


You are now able to create a map layer using JSON to configure the layer.
