ArcGIS Maps: Applying a camera to a scene view in ArcGIS Maps


Applying a camera to a scene view allows you to define a view based on location, heading, and altitude. One of the best use-cases for using a camera is to create a view that is displayed on the scene's initial load. This ensures the person viewing the scene can see a scenes data from an optimal position.

This article will guide you through several steps for applying a camera to an ArcGIS Maps scene layer.


  1. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.



  2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).



  3. In the Map Layers list, search by name for the map layer you desire to apply a picture-marker symbol for displaying the layer's features.



  4. Click the desired map layer to open the map layer's configuration form.

  5. In the Configuration text box, view the JSON used to construct the map layer.

    JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.

  6. Navigate the Configuration text box to view the title, and dataType parameters.

    The viewType parameter represents the type of the layer that will be displayed in the ArcGIS Maps application. The title parameter represents the name of the layer that will appear in the LayerList widget.



  7. Navigate the Configuration text box to view the JSON camera object. To configure the camera, you need to specify the position, heading, and tilt parameters. The position parameter represents the x, and y position of the camera. Heading represents he direction of the camera in degrees. Tilt represents the tilt angle of the camera in degrees in relation to the grounds surface.

  8. Click Update to apply the camera to the scene layer.
