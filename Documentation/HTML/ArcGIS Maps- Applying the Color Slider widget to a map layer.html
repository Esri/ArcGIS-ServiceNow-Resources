ArcGIS Maps: Applying the Color Slider widget to a map layer


This article will guide you through several steps for applying the Color Slider widget to an ArcGIS Maps map layer.


  1. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.



  2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).



  3. In the Map Layers list, search by name for the map layer you desire to apply Color Slider widget.


  4. Click the desired map layer to open the map layer's configuration form.


  5. Navigate the map layer's form to the Widget Information tab.


  6. In the Configuration text box, view the JSON used to construct the map layer.


    JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.


  7. Navigate the Configuration text box to where you would like to implement the widget. For this example, the code will be placed below the layer's visible parameter.


  8. The Color Slider widget uses visual variables to set stops on the slider widget.



Visual variables allow symbols to change dynamically based on values contained in a numeric attribute field. There are four types of visual variable renderers that can be used to render features. These types include color, opacity, rotation, and size. The example below uses the color type to dynamically change the symbol color based on the feature's numeric attribute value in the count field.



Visual variables are grouped into ranges of data values called stops. Between two and six stops can be applied in ascending order to the stops array. Each stop defines the mapping of values returned from the field to symbol colors. Once the visual variables are defined, the color slider widget can be applied in the map layer's JSON configuration.


  1. Navigate the Configuration text box to view the JSON widgets object. To configure the color slider widget, you need to specify the type of widget to create, and the min and max parameter values that are required for slider to apply changes to the symbols.

    The widget type is smartMapping/ColorSlider. To properly config the ColorSlider widget, you need to specify the min and max properties, primaryHandleEnabled, handlessSyncedToPrimary, and expand parameters.


    The primaryHandleEnable parameter if set to true, enables three handles in the widget instead of two. The third handle controls the position of the other two handles when modified. The handlessSyncedToPrimary parameter updates the outside handle positions when the primary handle is moved.


  2. When the widget parameters are configured in the Configuration text box, click Update to apply the changes to the map layer.


You are now able to apply a Color Slider widget to an ArcGIS Maps map layer. Download the ColorSlider widget JSON file.
