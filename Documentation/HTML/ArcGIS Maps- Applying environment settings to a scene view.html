ArcGIS Maps: Applying environment settings to a scene view


The ArcGIS Maps application supports applying environment settings to scene views. Environments settings enables settings specific to environment like atmosphere, lighting, and background to be applied to a scene.


This article will guide you through several steps for applying environment settings to a scene view.


  1. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.



  2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).


  3. In the Map Layers list, search by name for the map layer you desire to apply environment settings.



  4. Click the desired map layer to open the map layer's configuration form.

  5. In the Configuration text box, view the JSON used to construct the map layer.

    JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.

  6. Navigate the Configuration text box to view the title, dataType, and table parameters.



    The viewType represents the type of view to be displayed in the ArcGIS Maps application. The title parameter represents the name of the layer that will appear in the ArcGIS Maps Layer List.


  7. Navigate the Configuration text box to the location where you would like to configure the environment settings. For this example, to configure the environment settings, you need to specify the background parameter.

    The background property specifies the scenes background color. By setting the type to color specifies the type of background to be applied is an RGB color. The starsEnabled and atmosphereEnabled properties are configured using Boolean parameters. For this example, the stars and atmosphere enabled properties are set to false to allow the background to reflect the RGB color value.
