ArcGIS Maps: Applying a visual variables renderer to a map layer


ArcGIS Maps visual variables allow you to define a feature's visual representation based on numeric values. Size, color, opacity, and rotation are all values that can be defined when applying visual variables to a renderer. Visual variables can be applied to simple renderers and unique value renderers.

This article will guide you through several steps for applying visual variables to an ArcGIS Maps map layer.

  1. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.



  2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).


  3. In the Map Layers list, search by name for the map layer you desire to apply a picture-marker symbol for displaying the layer's features.



  4. Click the desired map layer to open the map layer's configuration form.


  5. Navigate the map layer's form to the Widget Information tab.


  6. In the Configuration text box, view the JSON used to construct the map layer.

    JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.


  7. Navigate the Configuration text box to view the title, dataType, and table parameters.

    The title parameter represents the name of the layer that will appear in the ArcGIS Maps Layer List. The dataType parameter represents the ArcGIS Maps data type that will be used to represent the layer. The table parameter references the name of the table in the CMDB.


  8. Navigate the Configuration text box to view the JSON symbol object. To configure the symbol, you need to specify the type of symbol to create, and the values that are required for configuring the symbol.

    The symbol type is simple-marker. To properly config a simple-marker symbol you need to specify the style, color, size, outline color, and outline width parameters.


    The map layers symbol has now been applied to the layerâ€™s configuration JSON. This is the symbol that will be used by the visual variables renderer to display features on the map.


  9. Now that the symbol has been applied to the configuration JSON, additional JSON needs to be applied specifying how the visual variables renderer is to use the previously defined symbol to visually change the size of the symbol based on the views scale.


    The following steps layout the process for creating a visual variable.


    1. Create the renderer.


    2. Specify the renderer type as visualVariables.


    3. Specify the visual variable renderer type. For this renderer, the type is size. Other variable types include color, opacity, and rotation.

    4. Create a value expression. For this renderer, the renderers symbology will update based on the map views scale.

    5. Create an array of stops. Stops apply changes to the symbols based on the stops value. For this renderer, the value represents a scale value. Based on the scale value, the visual renderer changes the symbols size when the view's scale value falls within stop value.


  10. When the symbol and visual variable renderer JSON parameters are configured in the Configuration text box, click Update to apply the visual variables renderer to the map layer.

Using this article, you navigated to a map layer's configuration parameters and applied a simple marker symbol and visual variables renderer to an ArcGIS Maps Map Layer.
