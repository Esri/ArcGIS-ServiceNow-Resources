ArcGIS Maps: Creating an ArcGIS Map as a scene


This article will guide you through several steps for creating an ArcGIS Maps scene map layer to display features in 3D.


  1. In the Service Now Filter Navigator enter Map Layers to view the ArcGIS Maps Map Layers item.



  2. Click the Map Layers result item to view a list of all the map layers available in your Service Now Personal Developer Instance (PDI).



  3. In the Map Layers list, click New to open the form for creating a new map layer.


  4. Enter a unique name for the map layer.


  5. Click Active to make the map layer table available in the tables directory and the Access Control Lists.


  6. For the Category field, enter ArcGIS Map.


  7. For the Subcategory field, enter Core Technology Services.



  8. Navigate the map layer's form to the Layer Configuration tab.


  9. In the Configuration text box, you will enter the JSON used to construct the map layer.

    JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. The JSON in the Configuration text box stores the layer's configuration data required for displaying the layer in the ArcGIS Maps map.


  10. Begin constructing the map layer by entering the scene's source JSON parameters.



    The viewType is "scene". A scene represents a 3D view of locations on the earth. For this example, the zoom parameter is set to 4. This zoom level creates an extent capable of displaying a geographically large country. The center parameter is used to navigate the scene to a fixed geographic location on the map. The example above, contains an array of x and y decimal degree coordinate values.


  11. When the scene's parameters are configured in the Configuration text box, click Update to apply the configuration JSON to the map layer.


You are now able to create a scene using JSON to configure the layer.
