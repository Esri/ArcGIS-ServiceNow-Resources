ArcGIS Maps: Create a layer list CMDB Menu Action

 

1.  In the Service Now Filter Navigator enter CMDB Menu Actions to view the CMDB Menu Actions item.

 

 

 

2. Click the CMDB Menu Actions to view a list containing all CMDB Menu Actions.

 

 

 

3.  Click New to create a new menu action.

 

A new menu action form is displayed. The menu action form contains all parameters required for creating the menu action. To apply the menu action to an ArcGIS Map application, you will need to configure the menu action to a map layer, or other component using JSON. JSON which stands for JavaScript Object Notation is a language-independent data format used to store and transmit data. 

 

4. For Name enter Get layer geometry type.

 

5. For Widget Type select Layer List Action.

 

6. Ensure that the menu action is Active.

 

 

7. In the Information tab, navigate to the Script configuration text box.

The script text box will contain all JavaScript code used to implement the menu actions action.

 

8. Insert the following line of code in the Script textbox.

alert(listItem.layer.geometryType);

 

9. Click Save to save the menu action.

 

10. Click the Back button    to return to the CMDB Menu Actions list.

 

11. Search the Name field for the Get layer geometry type menu action. This can be done by entering *Get layer type in the text box under the Name field.

 

 

12. Verify the menu action is listed in the CMDB Menu Actions list.

 

 

The menu action is now ready to be implemented in an ArcGIS Map application.

 

14. Open the menu action again to obtain the action's sys_id parameter.

 

 

The sys_id parameter is used to reference the menu action from the ArcGIS Map application's configuration JSON.

 

 

13. Open a map layer's configuration form and navigate to the JSON configuration textbox.

 

14.  Insert the menu action into the actionSections array's array.

 

15. Save the map layer configuration.

 

You are now able to create a CMDB Menu Action and configure the action in a map layer.
